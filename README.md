
# Redrob AI Candidate Ranker

An AI-powered candidate ranking system that matches resumes against job descriptions using semantic search, NLP, feature engineering, and machine learning.

## Features

- Resume Parsing
- Job Description Parsing
- Semantic Matching using Sentence Transformers
- Candidate Ranking
- Explainable Scores
- CSV Submission Generation
